# Kyle Annett Portfolio

Single-page portfolio built with [Astro](https://astro.build), using content collections for structured section data and component-scoped styling for each section.

## Requirements

- Node.js `>=22.12.0`
- `pnpm`

## Scripts

Run from the project root:

| Command | Purpose |
| :-- | :-- |
| `pnpm install` | Install dependencies |
| `pnpm dev` | Start local dev server (`http://localhost:4321`) |
| `pnpm build` | Build production output to `dist/` |
| `pnpm preview` | Preview production build locally |
| `pnpm check` | Run Astro type/content checks |
| `pnpm format` | Format Astro/TS/JS/JSON/MD with Prettier |
